# Paper figures

Each script here regenerates one figure from the paper out of a result file
produced by the runners in `analysis/`. They use a shared `paper_style.py` for
consistent fonts/ticks/DPI.

**Pending finalization against the published paper.** The exact set of figures
(and the numbers each one quotes) is being pinned to the final manuscript before
the corresponding scripts are ported here, de-hardcoded, and listed in a
figure → script → number table.
